# Назначение
> [!ABSTRACT] INFO
> Репощиторий **client** является имитацией микроконтроллера посылающий данные по протоколу UDP на шлюз



> [!ABSTRACT] Суть
> **asyncio.Event** — это простейший примитив синхронизации. Он работает как «флаг» или «светофор»: состояние может быть либо «установлено» (True), либо «не установлено» (False). 
> Позволяет одной корутине ждать, пока другая не подаст сигнал.
